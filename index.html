Fish Mod Loader

The Fish Mod Loader (FML) is a simple mod manager for fish written in fish. It provides some functions for fish shell to manage FML mods.

Simple

FML only does one thing: Load mods.

All other things are not considered.

Transparent

All functions and variables FML used can be directly called or read in interactive shell.

All functions and variables have their document, even only 1 line.

Fit

FML skips error when loading mods. Instead, it will give a warning to user.

Note that we plan to set a flag for FML to stop loading any mod when facing errors.
